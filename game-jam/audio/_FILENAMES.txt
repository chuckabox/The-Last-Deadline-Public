DROP AUDIO FILES HERE — THE GAME WILL PICK THEM UP AUTOMATICALLY
================================================================

Folder layout:
  audio/
    music/   <- five looping stage tracks
    sfx/     <- one-shot sound effects

If a filename is missing on disk, the game just runs without that sound
(no crash). Add files at any time, restart the scene, they'll play.

You can rename or repath any of these in the Godot editor by selecting the
MusicManager / SFXManager autoload nodes and editing the @export dictionary
in the inspector — but using the names below means zero config.

----------------------------------------------------------------
